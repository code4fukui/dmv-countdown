# 発車カウントダウン - DMV

阿佐海岸鉄道の発車時刻をカウントダウンで表示するWebアプリです。GTFSデータを使って発車時刻を表示します。

## デモ
[https://code4fukui.github.io/dmv-countdown/](https://code4fukui.github.io/dmv-countdown/)

## 機能
- 出発駅と到着駅を選択
- 現在時刻からの発車までの残り時間を表示
- 次の5本の発車時刻を確認できる

## 必要環境
特に必要ありません。ウェブブラウザで動作します。

## 使い方
ブラウザで上記のデモページにアクセスし、出発駅と到着駅を選択してください。現在時刻から次の発車時刻までのカウントダウンが表示されます。

## データ・API
本アプリでは、[福井県のオープンデータ](https://www.pref.fukui.lg.jp/doc/dx-suishin/opendata/gtfs_jp.html)のGTFSデータを使用しています。
